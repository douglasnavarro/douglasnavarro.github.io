Image Grabber

The Image Grabber is a GUI tool for capturing box-regions of the screen. I decided to write this tool in my spare time as a QA intern because my colleagues and I were losing too much time on the process of taking screenshots for functional test automation.

Web of presence sensors

This project was made to record how people use a building. The web of presence sensors is a system of two components: presence detectors and a web server. The presence detectors use ultrassonic sensors to detect people passing across its range and periodically send its counts via WiFi to the web server. The web server stores the data and produces dashboard views from it.
